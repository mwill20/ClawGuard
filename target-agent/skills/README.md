# Custom Skills

All skills in this directory are **custom-written or audited line-by-line** before deployment.

No community skills from ClawHub are installed without full source code review. This is a deliberate security decision — see `../../../lessons/README.md` for documentation on why.

## Skill Development

Custom skills will be added here as we build them. Each skill will include:

- `SKILL.md` — Instructions for the OpenClaw agent
- Source code (if applicable) — audited, lean, no hidden calls
- `AUDIT.md` — Security review notes documenting what was checked
